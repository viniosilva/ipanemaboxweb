# Ipanema Box WEB

Customers and services management

## Requirements

- [Node >= v20.x](https://nodejs.org/docs/latest-v20.x/api/documentation.html)

## Built With

- [Typescript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)

## Install

```sh
npm install
```

## Running

```sh
npm run dev
```

## Tests

```sh
npm test
```
